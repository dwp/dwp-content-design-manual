Service manual guidance

Writing effective letters

Earlier this year, I worked with the Service Manual team to publish the first draft of writing effective letters. It includes advice on:

  • deciding whether you should send a letter
  • how to structure letters
  • changing existing letters

We'd love to get feedback from other teams sending letters, so we can iterate and improve the guidance.

Sending letters with GOV.UK Notify

GOV.UK Notify will print, pack and post your letters for you.

There are 2 ways to send a letter with Notify. You can:

  • create a reusable template in Notify
  • upload your own letter as a PDF file

Uploading your own letters is an API only feature at the moment, but we're designing a simple way to let users upload PDFs through the Notify interface. The first iteration of this feature is nearly ready and we plan to release it in the new year.
